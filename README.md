# ipl-data-analysis

An exploratory data analysis of IPL cricket matches using Python, Pandas, and Matplotlib.

## What this project covers 
- Which teams won the most matches across all seasons
- Top venues by matches hosted
- Does winning the toss actually help?
- Season-wise performance of top 3 teams

## Tools used 
- Python 
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset IPL 
Complete Dataset from Kaggle (2008–2024)
